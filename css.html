Desenvolvimento web

O que e desenvolvimento web

O desenvolvimento web incluiu criacao, codificacao e programacao de sites e seus respectivos elementos e linguagens. ele tambem involve integracao com sistemas e ferramentas, armazenamentos de bancos de dados, ciberseguranca, entre outros, com o objetivo de estruturar cada detalhe de uma pagina da internet.

O que faz um desenvolvedor web?

O desenvolvedor de sites, chamado de profissional de Dev ou Developer e o responsavel por criar todo o planejamento estrutural de paginas, utilizando linguagens de programacao como HTML, CSS, JavaScript, PHP, entre varias outras.
